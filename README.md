# 🚀 Astro + Sanity Starter

Este es mi primer proyecto explorando el ecosistema de **Astro** y **Sanity.io**. Es un template diseñado para ser ligero, rápido y fácil de escalar, ideal para entender cómo conectar un **Headless CMS** con un framework de islas de interactividad.

## ✨ Características
- **Astro 5.17**: Renderizado ultra rápido y SEO "out of the box".
- **Sanity CMS**: Gestión de contenido estructurado con esquemas personalizados.
- **Zero-JS by default**: Aprovechando la arquitectura de islas de Astro.
- **Dynamic Routing**: Generación automática de páginas basadas en el contenido de Sanity.
